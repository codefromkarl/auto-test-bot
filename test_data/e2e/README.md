# E2E测试数据说明

本目录包含闹海E2E黄金路径测试所需的测试数据文件。

## 目录结构

```
test_data/e2e/
├── README.md              # 本说明文件
├── cover.jpg              # 默认剧本封面图片
├── scripts/               # 测试剧本文件
│   ├── sample_script.txt   # 示例剧本内容
│   └── ...              # 其他测试剧本
├── characters/            # 角色测试数据
├── scenes/               # 场景测试数据
└── assets/               # 其他测试资源
```

## 文件说明

### cover.jpg
- 用途：E2E测试中的剧本封面上传
- 要求：JPG格式，建议尺寸 16:9 或 3:4
- 来源：可使用任意合适的封面图片

### scripts/sample_script.txt
- 用途：E2E测试中的剧本内容输入
- 格式：纯文本，包含场景描述
- 内容：关于哪吒闹海的现代改编故事片段

## 数据使用方式

E2E测试框架会自动读取这些测试数据：

1. **剧本创建阶段**：使用 `cover.jpg` 作为封面，`scripts/sample_script.txt` 作为内容
2. **角色创建阶段**：使用 `characters/` 目录下的角色配置
3. **场景创建阶段**：使用 `scenes/` 目录下的场景配置

## 自定义测试数据

如果需要使用自定义测试数据：

1. 替换相应文件内容
2. 修改 `config/e2e_config.yaml` 中的测试数据路径配置
3. 确保文件格式和命名符合要求

## 注意事项

- 测试数据文件应该稳定可靠，避免使用过大文件
- 剧本内容应该简洁清晰，便于AI分析
- 图片资源应该符合闹海系统的格式要求
