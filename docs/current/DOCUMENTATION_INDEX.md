## 🎯 使用建议

### 📖 优先使用
- ✅ **docs/current/** 目录下的文档为当前有效版本
- ⚠️ **docs/archived/** 目录下的文档为历史参考，不推荐使用

### 📊 文档版本识别
- **当前版本**：文档名无特殊后缀，位于 docs/current/
- **归档版本**：文档位于 docs/archived/，有历史说明
- **草稿版本**：临时文档，包含 temp/draft 字符

### 🔍 文档贡献指南
1. **新增文档**：在相应目录下创建，并更新索引
2. **修改文档**：保持原有结构和格式
3. **归档处理**：过时文档移到 archived 目录，并更新索引
