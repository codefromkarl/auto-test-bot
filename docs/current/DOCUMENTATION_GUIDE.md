## 🔍 文档查找技巧

### 按类型查找
```
# 查找架构相关文档
find docs/current/ -name "*architecture*" -o -name "*设计*"

# 查找迁移相关文档
find docs/current/ -name "*迁移*" -o -name "*RF*"

# 查找指南相关文档
find docs/current/ -name "*指南*" -o -name "*使用*"
```

### 按状态查找
```
# 查看所有当前有效文档
ls docs/current/

# 查看所有归档历史文档
ls docs/archived/

# 查找特定主题文档
grep -r "关键词" docs/current/
```

## ⚠️ 重要注意事项

### 文档有效性
- ✅ **优先使用 current/** 目录下的文档
- ⚠️ **archived/** 目录为历史参考，内容可能过时
- ❌ **不要使用已被标记为 *_archived.md** 的文档

### 版本识别
- **当前版本**：文档名无特殊后缀，位于 docs/current/
- **归档版本**：位于 docs/archived/，在文档名中注明
- **草稿版本**：临时文档会明确标记 temp/draft

### 更新频率
- **核心文档**：重大功能完成后及时更新
- **索引文档**：新增文档后24小时内更新
- **归档文档**：每季度检查和优化
